<h1 align="center">GPT Free Share</h1>

简体中文 / [English](./README.md)

OpenAI API 共享解决方案，支持 ChatGPT。

## 功能

- 自托管 ChatGPT GUI
- Markdown 支持：LaTeX、代码高亮等
- 响应式设计、明暗模式
- I18n：简体中文、英语

## 开发

1. 获取 OpenAI base URL、API Key 和 model
2. 将它们填入 .env
3. 运行 `npm install`
4. 运行 `npm run dev`

## 部署

1. 获取 OpenAI base URL、API Key 和 model
2. 将它们填入 .env
3. 运行 `docker-compose up -d`
